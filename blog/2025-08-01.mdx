This release improves event and group page consistency, adds better search engine visibility, enhances dark/light theme handling, and introduces an improved mobile navigation experience.

<!-- truncate -->

---

## What's New

### Mobile Navigation
New mobile menu bar for visitors browsing the platform on phones and tablets, making it easier to explore groups and events on the go.

### Improved Theme Support
Your theme preference (dark or light mode) is now saved across sessions and automatically follows your system settings. Theme loads cleanly without any flash of the wrong colors.

---

## Improvements

- Consistent URL handling across all event pages for reliable sharing and bookmarking
- Calendar subscription links now work correctly for adding events to your personal calendar
- Better search engine visibility with comprehensive sitemap support
- Improved accessibility with proper color scheme support

---

## Bug Fixes

- Fixed issue where creating a new event from a group page wasn't pre-selecting the correct group
- Resolved problem with event images not saving during event creation
- Fixed group events page to work properly with new features
