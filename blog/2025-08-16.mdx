This release delivers two major upgrades: a completely rebuilt authentication system for faster, more reliable logins, and a unified payment experience with the new Universal Payment Modal for all event RSVPs and group subscriptions.

<!-- truncate -->

---

## What's New

### Universal Payment Modal
All payments—event RSVPs, ticket modifications, and group subscriptions—now use a single, consistent interface with clear cost breakdowns and trust indicators.

### Improved Refund Handling
- Partial modifications now calculate fees accurately
- Full cancellations process refunds correctly
- Clearer refund amounts shown before confirming

---

## Improvements

### Authentication
- Faster logins with smart session caching
- No more getting stuck on loading screens after idle time
- Sessions stay active reliably when switching between tabs
- Instant UI updates after logging in

### Payments
- RSVP modal pre-fills your previous ticket selections
- Ticket limits are now enforced properly during checkout
- Payment history shows clear, readable invoice details
- Consistent payment experience across events and groups

### Performance
- Significantly reduced loading times for authentication
- Faster data retrieval throughout the platform

---

## Bug Fixes

- Fixed sessions getting stuck in loading state
- Resolved duplicate ticket records during checkout
- Fixed capacity validation issues for events
- Corrected fee calculation accuracy
