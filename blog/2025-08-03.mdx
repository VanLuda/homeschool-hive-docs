This release focuses on production readiness with full mobile responsiveness, consistent dark mode styling, and restored email notifications for a better experience across all devices.

<!-- truncate -->

---

## Improvements

### Mobile Experience
- My Events dashboard now displays properly on all screen sizes
- Group discovery page stacks correctly and remains usable on mobile
- Group detail pages adapt layout and navigation for smaller screens
- Your Groups management page is now optimized for mobile devices
- Landing page adjusts structure for both logged-in and visitor views on all devices

### Dark Mode
- Applied consistent dark mode styles across the entire application
- All pages now render properly with correct backgrounds, text, and component contrast

### Email Notifications
- All transactional emails are now sending reliably
- RSVP confirmations, waitlist promotions, and group invites are delivered as expected

---

## Bug Fixes

- Fixed mobile navigation header layout issues on small screens
- Resolved issue where group images in "My Groups" were missing or misaligned
