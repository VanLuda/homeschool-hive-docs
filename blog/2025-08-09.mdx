This release delivers consistent page layouts, improved registration and password flows, enhanced search engine visibility, faster loading times, and a more stable authentication experience. Tab switching no longer causes unnecessary loading states.

<!-- truncate -->

---

## What's New

### Improved Search Engine Visibility
Events and groups now appear better in search results and social media shares:
- Events show rich information including dates, pricing, and images when shared
- Groups display location and membership details in search results
- Private groups are automatically hidden from search engines
- Better previews when sharing links on Facebook, Twitter, and LinkedIn

### Enhanced Social Sharing
Share events and groups easily with calendar integration for Google Calendar, Outlook, and Yahoo Calendar. Share directly to WhatsApp and Telegram.

---

## Improvements

### Layout and Spacing
- Consistent header and page spacing across the entire platform
- Smooth transitions when dismissing announcements
- No more content jumping during page load

### Registration and Password
- Form fields now preserve your input if there's a validation error (except password for security)
- Clear password requirements shown during registration
- Better error messages that explain what went wrong

### Performance
- Faster page loads with optimized data fetching
- Authentication loads instantly without delays
- Reduced loading indicators for a smoother experience

### Authentication
- Sessions stay active reliably across browser tabs
- Switching tabs no longer triggers unnecessary loading states
- Better handling of network issues with automatic recovery
