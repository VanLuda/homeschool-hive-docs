This release focuses on quality-of-life fixes for event creation and checkout, including improved ticket handling and more accurate fee calculations.

<!-- truncate -->

---

## What's New

### Add-On Tickets
Users can now purchase add-on tickets (materials, extras, etc.) along with their regular event tickets. Each ticket type now has its own independent capacity limit for better event management.

---

## Improvements

- Improved event creation flow with better navigation after creating an event
- Enhanced cancel RSVP modal for a clearer user experience
- Better transaction records for payment tracking

---

## Bug Fixes

- Fixed issue where group cards were not displaying upcoming event counts correctly
- Resolved problem with fee calculations during checkout
- Fixed refund amount calculations for canceled registrations
- Corrected issue where clicking "Add Ticket Type" or "Add Extra" was submitting the form instead of adding new options
- Fixed add-on ticket information not being saved during event creation
- Improved spacing in the Report a Problem modal
